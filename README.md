# P5-VUe.js Xayona

## Netlify: https://xayona.netlify.app/

### 1. Principles of UX

- 1.1. Avoid loading the interface with irrelevant content or bright colors
- 1.2. Use short descriptions for links, buttons, menus and headings.
- 1.2. Uses simple language appropriate to the sector or area of ​​the web
- 1.3 Uses intuitive visual language (icons, placement of elements) that makes it easier for the user to understand
- 1.4. Always give him a chance to escape a situation (like checking out a purchase in which he wants to change a product)
- 1.5. Works with standard web components
- 1.6. Make navigation intuitive
- 1.7. Leads the user in the processes they perform. Example: Checkout of a purchase

### 2. Accessibility Techniques

- 2.1. The links must be distinguishable
- 2.2. When hovering, the pointer will be a finger instead of the typical arrow
- 2.3. The links react when they pass over (hover) with: Color change, underline, visual effect.
- 2.4. Use only one `<h1>` per page to indicate the most important idea
- 2.5. Titles sections and subsections in order of importance, with `<h2>`, `<h3>` etc.
- 2.6. Users, including those with visual disabilities, are able to see the contents with clarity
- 2.7. Using Headings
